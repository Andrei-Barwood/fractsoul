# D71 - Score de severidad de anomalia

Fecha: 2026-03-24

## Entrega
- Score unificado `severity_score` (0-100) combinando detectores.
- Etiqueta de severidad:
  - `low`, `medium`, `high`, `critical`.

## Criterio de aceptacion
Cada reporte de anomalia retorna score numerico + nivel textual consistente.
