# D61 - Definir objetivos S3 (J/TH, anomalias, reporte diario)

Fecha: 2026-03-24

## Objetivo S3
Evolucionar de monitoreo operativo a analitica accionable:
- eficiencia energetica normalizada (J/TH) por equipo/rack/sitio,
- deteccion temprana de anomalias termicas y de degradacion,
- salida explicativa y recomendacion operativa,
- base para reporte diario automatico.

## Criterios de exito S3 (tramo D61-D75)
1. API expone eficiencia por equipo/rack/sitio con compensacion ambiente.
2. Motor de anomalias genera score, causa probable y recomendaciones v1.
3. Validacion con escenarios simulados etiquetados en pruebas.
4. Demo tecnica repetible para mostrar hallazgos y acciones sugeridas.
